# Neo-Futuristic Scroll-Driven Hero Animation

A premium, high-performance scroll-driven hero section built to demonstrate advanced frontend animation capabilities.

## Features
- **GSAP ScrollTrigger**: Buttery smooth scroll-linked animations.
- **Dynamic Masking**: Car acts as a moving mask, revealing a striking neon green track and headline.
- **Glassmorphism Metrics**: Beautiful, translucent statistics cards with hover-glow effects and spring physics.
- **Procedural Ambient Particles**: Hardware-accelerated (`will-change-transform`) floating particles for depth and immersion.
- **Next.js & TailwindCSS**: Built on a modern, ultra-fast tech stack.

## Tech Stack
- Next.js 15 (React 19)
- Tailwind CSS v4
- **GSAP (GreenSock Animation Platform)**
